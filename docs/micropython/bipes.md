# BIPES
{: .no_toc }

BIPES is on online blocks-based MicroPython editor, based on the open source project.

This version has been customised to meet the needs of the projects we are running at Sha Tin College.

* [Access BIPES here](https://pbaumgarten.com/bipes/)
