## Part 3: Add a Nested Loop

A **nested loop** is a loop inside another loop. The inner loop finishes completely before the outer loop does its next iteration.

**Example:**
```
REPEAT 3 TIMES {            ← outer loop (runs 3 times)
    REPEAT 2 TIMES {        ← inner loop (runs 2 times EACH outer iteration)
        Clap
        Stamp
    }
    Jump
}
```
*Total actions: outer runs 3 times × (inner runs 2 times × 2 actions + 1 Jump) = 3 × 5 = 15*
