## Flowchart Shape Key

| Shape | Name | Used for |
|-------|------|---------|
| Oval / Rounded rectangle | Start / End | The beginning and end of the process |
| Rectangle | Action | Something that happens (e.g., score = score + 5) |
| Diamond | Decision | A yes/no question (e.g., Is key = True?) |

**Diamond rule:** Every diamond has exactly TWO exits — one for YES, one for NO.
