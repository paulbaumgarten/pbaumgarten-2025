# Maze Grid — 5×5 Printable

**Print at A3 for best results. One per group.**

**Setup:** Before the lesson, cut trigger cards and place one face-down on each numbered square (except START and EXIT).

---

**Movement rules:**
- You can move UP, DOWN, LEFT, or RIGHT — one square per turn
- You cannot move diagonally
- On each square (except START): flip the trigger card and follow its rule
- Reach EXIT (E5) to win

**Starting values:** SCORE = 0, HEALTH = 10, KEY = False

---

## Legend

| Symbol | Meaning |
|--------|---------|
| → | Start here, facing right |
| ★ | Exit — reach here to win |
| [card] | A trigger card is placed face-down here |

---

*Teacher note: not all squares need a trigger card. You can leave some squares empty for faster play. Suggested route for demo: A1 → B1 → C1 → C2 → C3 → D3 → D4 → E4 → E5*
