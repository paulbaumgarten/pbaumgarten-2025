# Variable Labels — Print and Cut

**Instructions:** Print this page and cut out the three label cards. One set per pair.
Use during the starter activity — volunteers hold their card up so the class can see the current value.
Laminate for reuse across multiple classes.

---

**Starter Activity Instructions:**
1. Give one card to each of three volunteers.
2. Write the starting values in the boxes: HEALTH=10, GOLD=0, ITEMS=[]
3. Teacher reads each event aloud.
4. The volunteer whose variable changes crosses out the old value and writes the new one.
5. Hold the card up so the class can see the update happen in real time.
