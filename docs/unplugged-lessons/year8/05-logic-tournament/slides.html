Year 8 Computer Science — Boolean Logic Applied

The Rules

  • Pick one character from the grid — keep it secret
  • Your partner asks yes/no questions about ATTRIBUTES (not names!)
  • After each answer, cross off eliminated characters
  • First to correctly identify partner's character wins
  • Count your questions — fewest wins!

AND Queries

Combine 3 attributes into one query:

Hair = blonde AND Glasses = Yes AND Hat = No

How many characters match? Count them.

Is this a useful opening query? Or is it too specific?

Key Takeaway

Efficient Boolean queries halve the search space each time.

That's binary search — and it's how databases filter millions of records instantly.

Every SQL WHERE clause is a Boolean query.
