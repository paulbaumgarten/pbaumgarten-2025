# Number Cards — Sorting Network (1–6)

**Print and cut. Each walker holds ONE card throughout the activity.**

Cards should be large enough to hold up clearly — ideally A5 size (half an A4 sheet each).

**Instructions for walkers:**
- Stand at your starting position (teacher assigns randomly shuffled cards)
- At each layer, if you are part of a comparator pair: the person with the LOWER number moves to/stays in the LEFT lane; HIGHER number moves to the RIGHT lane
- All pairs in a layer move SIMULTANEOUSLY

**Print several sets to allow re-runs with different starting orders.**
