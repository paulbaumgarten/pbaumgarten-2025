# Package your IA into the compliant ZIP file
{: .no_toc }

1. Download and rename the [template file](ia-template.zip) from abc123.zip to your 6 digit IB code.
2. Unzip the file into a folder
3. Replace the placeholder PDF files with PDFs for each of your criteria. Make sure the file names match the template ones exactly. Do the same with your video file.
4. Copy your program files into the Product folder
5. Open the HTML file into a text editor.
   * Put your 6 digit code in the "Student name" section.
   * Change line 55 of the HTML file with any instructions required for the marker to install & run your project
   * Leave all other parts of the HTML file unchanged.
6. Double click the HTML file to open it into the web browser, check the links to the PDF files work.
7. Re-zip the folder back into a ZIP file named after your 6 digit code
8. Submit.
